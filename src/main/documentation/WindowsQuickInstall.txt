Polyglot Quick Install (Windows):

The following ports will be needed: 8182, 8183, 50000, 50001, 50002

1. Install Windows and the software you wish to use.

2. Install Java and AutoHotKey.

3. Download Polyglot snapshot from http://isda.ncsa.illinois.edu and unzip.

4. Open command prompt to this folder and run the following to configure and install ImageMagick script:

> ScriptDebugger -config scripts/ahk/ImgMgk_convert.ahk

5. Edit SoftwareServer.conf:

AHKScripts=scripts/ahk-configured

6. Edit SoftwareServerRestlet.conf and uncomment second line:

DistributedServer=localhost:8183

7. Run SoftwareServerRestlet.bat

8. From a browser go to localhost:8182 to see if the Software Server is working.

9. Edit PolyglotServer.conf and comment out web interface:

#PolyglotWebInterface=true

10. Run PolyglotServer.bat.

11. Put some images in the data folder.

12. Run PolyglotPanel.bat.
